# 🌸 She Can Foundation - Empowering Girls & Women Globally

A modern, responsive, and accessible landing page designed for **She Can Foundation**, an organization dedicated to empowering girls and women through education, skill development, mentorship, and career opportunities.

## 📖 Overview

The She Can Foundation website serves as a digital platform to showcase the organization's mission, inspire visitors, and encourage community participation through donations and membership programs.

The project focuses on:

* Clean and modern user interface
* Responsive design across devices
* Accessibility-friendly components
* Dark/Light theme support
* Strong visual storytelling
* Performance-focused front-end architecture
